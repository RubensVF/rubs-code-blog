## Crear el proyecto de maven

Como primer paso creamos un proyecto maven a partir de un arquetipo que ya nos prove el grupo Apache Maven.
Para crear el proyecto ecribims el siguiente comando desde la terminal.

```cmd
mvn archetype:create
  -DgroupId=com.chuidiang.ejemplos.plugin_maven
  -DartifactId=plugin_maven
  -DarchetypeArtifactId=maven-archetype-mojo
```

Una vez creado lo abrimos en nuestro editor favorito.
